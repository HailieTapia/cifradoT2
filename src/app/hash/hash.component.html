Instrucciones de SHA-2:

  1. Ingresa el mensaje que deseas cifrar o descifrar.
  2. Decide qué variante de SHA usar (SHA-224, SHA-256, SHA-384 o SHA-512).
  3. Presiona el botón de 'Cifrar' o 'Descifrar' para obtener el resultado.

Ejemplo: Para el mensaje "HOLA MUNDO" podría convertirse en un hash que se vea algo así:
